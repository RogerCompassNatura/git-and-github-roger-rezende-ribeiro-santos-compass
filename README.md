## Dicionário GIT do QA
| Comando | Entendimento                                                                                             |
|:--------|:---------------------------------------------------------------------------------------------------------|
| Commit  | Quando ocorre um registro das alterações realizadas em um projeto versionado                             |
| Push    | Quando ocorre o envio de um commit do computador do usuário para um repositório remoto                   |
| Pull    | Quando ocorre o envio das atualizações no repositório remoto para o computador do usuário                |
| Merge   | Quando organiza as alterações de diferentes branches para uma única                                      |
| Branch  | É uma versão paralela do projeto dentro do mesmo repositório para trabalhar em mudar o projeto principal |

## Comandos favoritos
Os comandos que mais utilizei em todo aprendizado no GIT foram:
- Commit
- Push
- Branch
